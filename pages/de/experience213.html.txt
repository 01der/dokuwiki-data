====== [hemmerling] Übersetzen von in Software-Quelltexten befindlichen deutschen Kommentaren in die englische Sprache ======
===== Projekt-Auftrag =====
  *Übersetzen von in Software-Quelltexten des OpenSource Office-Paket [[http://de.libreoffice.org/|Libre Office]], einer Software für Arbeiten im Büro, befindlichen deutschen Kommentaren in die englische Sprache, im Rahmen der Veranstaltung *Veranstaltung "LibreOffice Hamburg HackFest 2013" ( [[http://wiki.documentfoundation.org/Hackfest/Hamburg2013|The Document Foundation "LibreOffice Hamburg HackFest 2013"]], [[http://www.facebook.com/events/455098597892904/permalink/455098601226237/|Facebook "LibreOffice HackFest"]], [[http://blog.attraktor.org/2013/06/libreoffice-hackfest/|Attraktor Blog "LibreOffice HackFest"]] ), 2013-06-15 - 2013-06-16.
===== Projekt-Auftraggeber =====
  *[[http://www.documentfoundation.org/|The Document Foundation]].
===== Projekt-Rolle / Eigene Aufgabe im Projekt =====
  *Technischer Redakteur, Übersetzer.
===== Eigene Tätigkeiten im Projekt =====
  *Übersetzen von in Software-Quelltexten befindlichen deutschen Kommentaren in die englische Sprache.
===== Im Projekt eingesetzte Tools =====
  *Text-Editor ( Eclipse ).
===== Projekt-Ergebnis =====
  *In 2 Quelltext-Dateien der OpenSource Office-Paket [[http://de.libreoffice.org/|Libre Office]], einer Software für Arbeiten im Büro, sind die deutschen Kommentare ins englische übersetzt.
    -[[http://cgit.freedesktop.org/libreoffice/core/commit/?id=98b4c3303e95fa076acc2ebaf195346f7545d107|index: libreoffice/core. main, development code repository - Age: 2013-06-15, Commit message "Translate German comments", author: "Rolf Hemmerling", file "a/include/svx/svdmrkv.hxx"]].
    -[[http://cgit.freedesktop.org/libreoffice/core/commit/?id=3f1fecdbc784ab2c98321b7de5edfedef690c77c|index: libreoffice/core. main, development code repository - Age: 2013-06-17, Commit message "Translate German comments", author: "Rolf Hemmerling", file "a/sw/source/core/text/itrcrsr.cxx"]].
===== Projekt-Erfolg =====
  *Das OpenSource Office-Paket [[http://de.wikipedia.org/wiki/LibreOffice|Libre Office]] stammt vom in Deutschland in den 1990er Jahren entwicklelten kommerziellen Office-Paket [[http://de.wikipedia.org/wiki/StarOffice|StarOffice]] ab. 
  *Die aus den 1990er Jahren stammenden deutschen Kommentare der deutschen Software-Entwickler in den Software-Quelltexten können ( erst ) nach der Übersetzung is englische den internationalen, in der [[http://wiki.documentfoundation.org/|The Document Foundation]] organisierten und meist nicht deutschsprachigen OpenSource Software-Entwicklern helfen, die Software-Quelltexte zu verstehen, als Voraussetzung für eine erfolgreiche Weiterentwicklung des Produkts.
  *Durch die Übersetzungs-Mithilfe von mir, Rolf Hemmerling, habe ich dazu ein kleines Stück beigetragen.
