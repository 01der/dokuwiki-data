===== Vortrags-Thema =====
  *Der Microsoft SQL Server 2005, Developer Edition steht Studenten der Fakultäten I, II und IV der FH Hannover über das MSDNAA Programm von Microsoft kostenlos zur Verfügung. Im Fach „Business Intelligence“ im WS2008/2009 wurde nun zwei unterschiedlichen Studenten-Teams die Gelegenheit gegeben, die Business Intelligence ( BI) -Fähigkeiten des Produkts den anderen Studenten vorzustellen. Ein Teil der Studenten hat im WS2008/2009 auch die ergänzende Vorlesung „Data Mining“ gehört, in der die wichtigsten Data-Mining Algorithmen erklärt wurden, die auch SQL Server 2005 implementiert hat.
  *Die Lernziele der Präsentation und des Berichts sind das Vermitteln von  Produkt-Grundkenntnissen über Microsoft SQL Server 2005, Einsatzszenarien von Microsofts BI-Produkten sowie den Workflow einer BI-Abfrage mit „SQL Server Business Intelligence Studio“. Zum anderen sollten die Zuhörer der Präsenation und Leser des Berichts die in Vorlesungen vermittelte BI- und DataMining-Theorie in einer einfachen BI-Abfrage mit dem „SQL Server Business Intelligence Studio“ umsetzen können.
