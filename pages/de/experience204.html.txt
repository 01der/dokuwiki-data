====== [hemmerling] Microsoft Windows Phone Anwendung "Hemmerling(TM) Phone" ======
===== Projekt-Auftrag =====
Erstellung der Microsoft Windows Phone Anwendung "Hemmerling(TM) Phone" ( 2010-11-15 - 2010-11-21 ).
===== Projekt-Aufgaben =====
Erstellung einer einfachen Windows Phone Anwendung mit Software-Werkzeugen von Microsoft, durch Konfigurieren bzw. Nutzung von GUI-Builder Funktionen, ohne Erstellung von C# Software-Quelltexten, d.h. ohne Notwendigkeit zur C# Programmierung.
===== Eigene Tätigkeiten im Projekt =====
  *Basis-Schulung durch Besuchen des Vortrags "Windows Phone 7 – Das portable Tor zur Welt" von [[http://blogs.msdn.com/twendel|Tom Wendel]] auf dem von Lars Keller, [[http://www.netcreate.de|netcreate multimediaentwicklungen OHG]] geleiteten Regionaltreffen der [[http://www.dotnet-braunschweig.de|.NET Developers Group Braunschweig]] in Braunschweig, Deutschland ( 2010-11-15 ).
  *Erstellung und Pflege der Website-Seite [[:en:windowsphone.html|Windows Phone]].
  *Erstellung der Software-Anwendung.
===== Im Projekt eingesetzte Tools =====
  *[[http://www.microsoft.com/expression/windowsphone/|Microsoft Expression Blend 4 for Windows Phone]].
  *[[http://en.wikipedia.org/wiki/Microsoft_Visual_Studio|Microsoft Visual Studio]] Express 2010.
  *[[http://www.microsoft.com/downloads/en/details.aspx?FamilyID=04704acf-a63a-4f97-952c-8b51b34b00ce|Microsoft Download Center "Windows Phone Developer Tools RTW"]].
    *[[http://www.microsoft.com/downloads/en/details.aspx?FamilyID=49b9d0c5-6597-4313-912a-f0cca9c7d277|Microsoft Download Center "Windows Phone Developer Tools October 2010 Update"]].
