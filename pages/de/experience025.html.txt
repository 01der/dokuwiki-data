====== [hemmerling] Konfiguration von PC-Hardware, Installation von Betriebssystemen und Anwendungen, Installation von Novell und Windows Netzwerken ======
===== Projekt-Auftrag =====
Konfiguration von PC-Hardware, Installation von Betriebssystemen und Anwendungen, Installation von Novell und Windows Netzwerken ( 1991 - ).
===== Projekt-Auftraggeber =====
  *Kunden ( Privatleute, Freiberufler, Gewerbetreibende), die einen gebrauchten, voll funktionstüchtigen PC kaufen wollen.
  *Mittelständischer Bauunternehmer.
===== Projekt-Aufgaben =====
  *Installation von Betriebssystemen, Treibern und Anwendungen.
  *Konfiguration von PC-Hardware.
  *Installation von Novell und Windows Netzwerken. 
  *EDV-Beratungtätigkeit. 
  *Telephonische Kundenbetreuung.
===== Im Projekt eingesetzte Tools =====
PC, PC-Karten, Betriebssysteme, Software.
