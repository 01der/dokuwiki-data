===== Installing Linux on Asus EEE 901 =====
==== Configuration ====
  *Asus EE 901, BIOS 
    *"Boot / Boot Settings - Boot Device Priority".
      *"1st Boot Device": "[USB:USB USB Flash ]".
    *"Boot / Boot Settings - Hard Diks Drives".
      *"1st Drive - [USB: USB USB Flash]".
      *"2nd Drive - [HDD:SS-ASUS-PHISON]" -> 8 Gbyte size.
      *"3rd Drive - [HDD:SM-ASUS-PHISON]" -> 4 GByte size.
