===== Some other Non-SDL Tools =====
  *The commercial [[http://www.eventhelix.com/|EventHelix "EventStudio System Designer"]] for Windows - free 45 days trial version. It can be used to design call flow diagrams ( e.g. an "ISUP to ISUP call" ), which is not part of the SDL language, but which is used in system design for telecommuncations.
