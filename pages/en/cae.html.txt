===== Finite Element Method ( FEM ) / Finite Element Analysis ( FEA ) and FEM for Design of Electric Engines - Software =====
==== Free Tools ====
  *[[http://www.calculix.de/|CALCULIX: A Three-Dimensional Structural Finite Elemente Program]], [[http://www.dhondt.de/|CALCULIX: A Three-Dimensional Structural Finite Elemente Program]], [[http://www.bconverged.com/products.php|bConverged "bConverged CalculiX for Windows"]].
  *The OpenSource FEM / FEA tool "Computer Aided Learning of the Finite Element Method" ( [[http://calfem.sourceforge.net/|SourceForge "CALFEM"]], [[http://www.sourceforge.net/projects/calfem/|SourceForge "CALFEM"]] ) - "An interactive computer program for teaching the finite element method (FEM)".
  *The OpenSource tool [[http://www.DAEdalon.org/|Prof. Dr.-Ing. Herbert Baaser, Solid Mechanics, TUD "DAEdalon"]] for use with [[http://www.matlab.de/|MATLAB]] 5 and 6.
    *Book [[http://www.amazon.de/exec/obidos/ASIN/3642131522/hemmerling-21|Herbert Baaser "Development and Application of the Finite Element Method based on MatLab"]].
    *Accompanying website for the book [[http://www.springer.com/materials/mechanics/book/978-3-642-13152-3|Springer Verlag "Development and Application of the Finite Element Method based on MatLab. Development"]].
  *The OpenSource C++ library [[http://www.dune-project.org/|DUNE, the Distributed and Unified Numerics Environment]].
    *[[http://en.wikipedia.org/wiki/http://en.wikipedia.org/wiki/Dune_%28software%29|EN.Wikipedia "Dune (software)"]], [[http://de.wikipedia.org/wiki/Dune_%28Software%29|DE.Wikipedia "Dune (Software)"]].
  *The OpenSource FEM / FEA tool [[http://www.csc.fi/english/pages/elmer|CSC — IT Center for Science Ltd. "Elmer - Open Source Finite Element Software for Multiphysical Problems"]].
    *[[http://en.wikipedia.org/wiki/Elmer_FEM_solver|EN.Wikipedia "Elmer FEM solver"]], [[http://de.wikipedia.org/wiki/Elmer_%28Software%29|DE.Wikipedia "Elmer (Software)"]].
  *FEAP, FEAPpv.
    *The free FEM / FEA tool [[http://www.ce.berkeley.edu/projects/feap/feappv/|Civil and Environmental Engineering. University of California, Berkeley "FEAPpv A Finite Element Analysis Program: Personal Version"]].
    *[[http://www.ce.berkeley.edu/projects/feap/|Civil and Environmental Engineering. University of California, Berkeley "FEAP - A Finite Element Analysis Program "]].
  *The free FEM / FEA tool [[http://people.ee.ethz.ch/~femag/|Prof. Dr. K. Reichert "FEMAG"]] for Windows.
    *"Das FE-Programm für  die Analyse  und Simulation elektrischer Maschinen und Geräte".
    *The free demo version can handle 25.000 elements and expires after 2 years.
  *The OpenSource FEM / FEA tool [[http://felt.sourceforge.net/|SourceForge "FElt"]], [[http://www.sourceforge.net/projects/felt|SourceForge "FElt"]].
  *The free FEM / FEA tool "FEPC".
    *Book [[http://www.amazon.de/exec/obidos/ASIN/0072922362/hemmerling-21|David V. Hutton, David Hutton: "Fundamentals of Finite Element Analysis"]].
    *Accompanying website [[http://highered.mcgraw-hill.com/sites/0072520361/|Mechanical Engineering Design, 7/e - Information Center]], [[http://highered.mcgraw-hill.com/sites/0072520361/student_view0/|Mechanical Engineering Design, 7/e - Student View]].
    *[[http://highered.mcgraw-hill.com/sites/0072520361/student_view0/fepc_program.html|Download of the student edition of FEPC]].
  *The OpenSource FEM / FEA tools of the [[http://www.freefem.org/|freeFEM]] family of software.
    -freeFEM++.
    -freeFEM3D.
    *[[http://kfem.sourceforge.net/|SourceForge "KFEM"]], [[http://www.sourceforge.net/projects/kfem|SourceForge "K Finite Element Method"]] for KDE with OpenGL.
  *The OpenSource FEM / FEA tool [[http://support.sdtools.com/gf/project/openfem/|SDToolsGForge "OpenFEM"]], [[http://gforge.inria.fr/projects/openfem/|INRIAGForge "OpenFEM"]].
  *The OpenSource FEM / FEA tool [[http://pdnmesh.sourceforge.net/|SourceForge "pdnMesh"]], [[http://www.sourceforge.net/projects/pdnmesh|SourceForge "pdnmesh Finite Element program"]].
  *The free [[http://members.rediff.com/srk007/html/fea/main.html|Archive.org "A. Sankararamasubramanian, S. Sivaramakrishnan, S. Srinivasan K. Sriram: S4 Finite Element Analysis Software"]] ( - 2007-07-05 ).
    *[[http://wayback.archive.org/web/*/http://members.rediff.com/srk007/html/fea/download.html|Archive.org "A. Sankararamasubramanian, S. Sivaramakrishnan, S. Srinivasan K. Sriram: S4 Finite Element Analysis Software - Download"]] - Download refers to invalid server. I don´t know any current donwload site.
  *The free FEM / FEA tool [[http://z88.uni-bayreuth.de/|University of Bayreuth "Z88 Aurora"]].
==== Commercial Tools ====
  *The commercial [[http://www.ansys.com/|ANSYS, Inc. "ANSYS"]].
    *[[http://www.ansys.com/Ansys/de_de|ANSYS Germany GmbH]], [[http://www.cfx-germany.com/|ANSYS Germany GmbH]].
    *[[http://www.cadfem.de/produkte/ansys/academic/ansys-educational.html|CADFEM GmbH "ANSYS Educational (ED) Version]] - "Verwendet einen 'Klick wrap' Lizenzvertrag ohne die Notwendigkeit für einen separaten Lizenzschlüssel. Die aktuelle Version von ANSYS ED ist 10.0".
    *Free text editors which support syntax highlighting of ANSYS commands: See [[sohosoftware.html|Small Office & Home Office ( SOHO ) Software]].

    *[[http://en.wikipedia.org/wiki/COMSOL_Script|EN.Wikipedia "COMSOL Script"]] - "In Version 3.5a of COMSOL Multiphysics, Comsol Script was removed :-(. Interfacing COMSOL Multiphysics 3.5a from a script based environment now requires MATLAB from The MathWorks, Inc".

    *Once there was a limited "Fedem Free Edition 4.2.1". However, the "Finite element part" was not supported by the free edition.
