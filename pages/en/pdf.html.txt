    *[[http://www.adobe.com/support/downloads/product.jsp?platform=windows&product=10|Adobe Reader for Windows - Downloads]] - Download archive with legacy versions for legacy operating systems.
    *Adobe Reader 9.5.0 is the latest edition for W2k.
    *Adobe Reader 6.02 is the latest edition for Win98SE.
    *Tip for use of "Adobe Reader X" on Windows8.1 computer with touch display: You must disable "View / Show/Hide / Toolbar Items / [_] Touch Mode" to enable the display of the usual window scrollbar for desktop computers" :-).

  *The free [[http://www.tracker-software.com/product/downloads|Tracker Software Products "PDF-XChange Viewer"]] for Windows.

    *[[http://ghostscript.sourceforge.net/|SourceForge "The Ghostscript project"]], [[http://www.sourceforge.net/projects/ghostscript/|SourceForge "Ghostscript"]], [[http://www.ghostscript.com/download/|Ghostscript, GhostPCL, GhostXPS and MuPDF Downloads Page]] - Current versions of Ghostscript.

      *[[http://pages.cs.wisc.edu/~ghost/redmon/|The University of Wisconsin "RedMon - Redirection Port Monitor"]].
    *[[http://en.wikipedia.org/wiki/Ghostscript|EN.Wikipedia "Ghostscript"]], [[http://de.wikipedia.org/wiki/Ghostscript|DE.Wikipedia "Ghostscript"]].

===== Free PDF Printer Drivers for Windows =====
  *The free [[http://www.dopdf.com/|doPDF]] for WinXp. Does not make use of GhostScript.
    *"doPDF 7.0 build 330" ist the latest edition for W2k, while doPDF 7.3 just runs on WinXP.
  *The free [[http://www.pdfprinter.at/de/|eDocPrintPro]] for Windows with .NET 2.0, makes use of GhostScript 9.05.
  *The free [[http://www.freepdfxp.de/|FreePDF]] for Windows, a graphical user interface bundled with Ghostscript 9.05.
    *FreePDF 2.11 ( with GhostScript 8.14 ) is the latest edition for Win95, Win98SE. 
    *FreePDF 4.07 is the latest edition for W2k and isn't suitable for Win8.
    *FreePDF 4.08 and newer are just for WinXP and newer, including Win8.
  *The legacy [[http://www.schmitz-huebsch.net/gs4word/|Schmitz-Hübsch "Gs4Word"]] of 2006, a graphical user interface bundled with Ghostscript 8.53 - "Mit Gs4Word erzeugen Sie in Microsoft Word, Excel und Powerpoint mit einem Klick PDF-Dateien".
  *The free [[http://www.pdf995.com/|Pdf995 Suite]] by [[http://www.software995.com/|Software995]] for WinXP.
    *[[http://pdf995.com/archive.html|Pdf995 Suite - Archive]].
    *I don´t know the latest editions of Software995 tools for W2k, but the Software995 editions of 2008-10 still run on W2k.
  *The free [[http://www.primopdf.com/|Nitro PDF, Inc. "PrimoPDF"]] for Windows with .NET 2.0.
  *The OpenSource [[http://www.pdfforge.org/|PDFForge "PDFCreator - The free PDF converter tool"]], [[http://pdfcreator.sourceforge.net/|SourceForge "PDFCreator"]], [[http://www.sourceforge.net/projects/pdfcreator/|SourceForge "PDFCreator"]] for WinXP, a graphical user interface bundled with GhostScript. Shipped with the free "PDFArchitect".
    *The online manual [[http://www.pdfforge.org/content/pdfcreator-user-manual|PDFForge "PDFCreator User Manual"]].
    *PDFCreator 1.2.3 is the latest edition for W2k. Additionally, you might use the PDFArchitect of the installer "PDFCreator 1.3.2" on W2k.
    *If you don´t care, Adware will be installed with the software :-(.
  *The free [[http://www.pdf24.org/creator.html|PDF24 Creator]] for WinXP.
    *[[http://download.cnet.com/PDF24-creator/3000-6675_4-10813698.html|CNET "PDF24 Creator 2.01"]] is the latest edition for W2k.
  *The OpenSource [[http://qvpdf.sourceforge.net/|SourceForge "qvPDF"]], [[http://www.sourceforge.net/projects/qvpdf/|SourceForge "qvPDF"]] for Windows.
  *The free [[http://www.tinypdf.com/|TinyPDF]] for Windows. Does not make use of GhostScript.
  *The free [[http://www.tracker-software.com/product/downloads|Tracker Software Products "PDF-XChange Lite, non-commercial release"]] for WinXP.
    *[[http://www.tracker-software.com/free_lite_home.html|Tracker Software Products "Offer for Home & Academic Users"]] - "If you are a Home or Academic user - for a limited time we are offering PDF-XChange Lite 2012 for FREE".
  *Once in 2011, the commercial PDF driver "[[http://www.avanquest.com/Deutschland/|Avanquest]] PDF Experte Ultimate 7" was offered by PEARL Agency as budget offer. Nowadays, in 2012, if I install the driver and try to run it after installation, the urged authentification fails by the error message "Die von Ihnen für die Installation von ... ausgewählte Seriennummer ... wurde bereits verwendet oder Sie haben das Maximum der zulässigen Installationen erreicht. Daher kann die Software nicht mit dieser Seriennummer ausgeführt werden" :-(.
===== PDF 2 Postscript Converter =====
  *The DonationWare [[http://www.transcom.de/transcom/en/2004_pdf-t-maker.htm|Transcom. Ingenieurbüro Johannes Graubner "PDF-T-Maker"]] - "You want to produce rich PDF files which contain all meta information from your Word document as bookmarks, hyperlinks, comments etc.?", "Similar to almost any PDF converter, as first step PDF-T-Maker converts your Word document into a PostScript file. You can send this PostScript file through a distiller, that generates the PDF file from it. As distiller you may use stand-alone solutions such as Adobe Distiller or GhostScript as well as pre-configured, highly automated solutions such as FreePDF XP, qvPDF, PDFCreator etc.".
