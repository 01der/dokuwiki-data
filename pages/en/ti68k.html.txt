===== History and Motivation for this Page =====
  *Since January 2005, I own a new programmable calculator, the "Texas Instruments Voyage 200". Once I owned some other programmable calculators, like Texas Instruments TI-57, TI-59 and "Sharp PC1403H".
  *Texas Instruments "TI-89", "TI-89 Titanium", "TI Voyage 200" pocket calculators and the "CBL 2" are powered by a MC68000 CPU. You can write your own applications in BASIC programming language, plus binary applications in "C" and assembly language.

===== "Voyage 200" FAQ =====
  -How to transfer files to the "Voyage 200" device ?
    -Run the "TI Device Explorer" on the Windows host computer.
    -In the "TI Device Explorer", create an empty file "*.TIG" in the Windows filesystem.
    -With the "Windows Explorer" or by Desktop, click the just created file "*.TIG" for opening. It appears as a directory in the filesystem, in the "Windows Explorer" and on the desktop. But it appears to be a file in the DOS window. The "*.TIG" file is an archive file, which can be packed and deloaded from the Windows Explorer and the Desktop.
    -By Drag+Drop copy files into the "*.TIG" file.
    -Transfer the file by "T-Connect" to the "Voyage 200" device
    -On the "Voyage 200", the contents is stored in a folder with the name of the "*.TIG" file.
  -How to edit TI-Basic files on the Windows-PC ?
    -Call "TI-Graph Link 89".
    -The sofware does not know the file extentions "*.V2F" ( functions ) and "*.V2P" ( programs ). Therefore at "Open file" always enter the filename "*.*", and then select the requested file !
