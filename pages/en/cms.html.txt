==== WAMPs ====
    *[[http://www.apache.org/|Apache]].
       *[[http://httpd.apache.org/|Apache Server]].
       *[[http://www.apachefriends.org/en/xampp.html|Apache friends "xampp"]].
       *[[http://xampp.sourceforge.net/projects/|SourceForge "XAMPP"]], [[http://www.sourceforge.net/projects/xampp/|SourceForge "XAMPP"]].
         *Built-in MySQL administration: 
           *[[http://localhost/phpmyadmin/|localhost/phpmyadmin/]] for port 80.
           *[[http://localhost:8502/phpmyadmin/|localhost:8502/phpmyadmin/]] for port 8502.
         *To start MySQL as service, please run "C:\Program Files\MySQL\xampp\mysql_installservice.bat" ( using the "MySQL Console Panel" does not really work, somehow ).
         *XAMPP 1.7.3 is the latest version for W2k.
         *XAMPP 1.7.4 ( ( "VC6" is part of the names of archives and installer ) and XAMP 1.7.5 and newer ( "VC9" ist is part of the names of archives and installer ) are shipped with a version of MySQL which is compiled so that it just runs on WinXP, but not on W2k anymore :-(.
         *XAMPP 1.8.0 is shipped with PHP6. By default, it causes much warning messages with "standard" PHP5 code running bravefully on PHP5, e.g. old editions of [[dokuwiki.html|DokuWiki]] :-(.
         *Warning: 
           *Apache server of XAMPP is using both port 443 and a configurable port ( by default, port 80 ). 
           *The Skype phone application is using the ports 80 and 443:-(.
         *[[http://www.microsoft.com/en-us/download/confirmation.aspx?id=5582|Micosoft Download Center "Microsoft Visual C++ 2008 SP1 Redistributable Package (x86)"]].
         *[[http://forum.dokuwiki.org/post/21325|DokuWiki Forum "fatal errors: maximum execution time of 30 seconds exceeded"]].
           *The maximum execution time is defined by the setting "php_value max_execution_time 30" in the file "C:\Program Files\xampp\php\php.ini".
      *[[http://i.justrealized.com/2009/how-to-run-xampps-php-cli-and-cakephp-console/|i.justrealized "How to run XAMPP’s PHP CLI and CakePHP console"]] -> With PHP acccounts with commandline access, called "CLI binary" with XAMPP, there is no restriction to 30 seconds runtime of PHP commands.
    *[[http://www.easyphp.org/|EasyPHP]].
    *[[http://www.lighttpd.net/|lighttpd]] ( spoken "lighty" ) - Suitable and recommended for embedded systems on Linux :-).
    *[[http://www.nginx.org/|nginx]] - Suitable and recommended for embedded systems on Linux :-).
      *Book [[http://www.amazon.de/exec/obidos/ASIN/1782162321/hemmerling-21|Clément Nedelcu: "Nginx HTTP Server - Second Edition"]].
      *[[http://en.wikipedia.org/wiki/Nginx|EN.Wikipedia "Nginx"]], [[http://de.wikipedia.org/wiki/Nginx|DN.Wikipedia "Nginx"]].
