           *[[http://localhost:8502/phpmyadmin/|localhost:8502/phpmyadmin/]] for port 8502.

         *XAMPP 1.7.3 is the latest version for W2k.
         *XAMPP 1.7.4 ( ( "VC6" is part of the names of archives and installer ) and XAMP 1.7.5 and newer ( "VC9" ist is part of the names of archives and installer ) are shipped with a version of MySQL which is compiled so that it just runs on WinXP, but not on W2k anymore :-(.
         *XAMPP 1.8.0 is shipped with PHP6. By default, it causes much warning messages with "standard" PHP5 code running bravefully on PHP5, e.g. old editions of [[dokuwiki.html|DokuWiki]] :-(.
         *Warning: 
           *Apache server of XAMPP is using both port 443 and a configurable port ( by default, port 80 ). 
           *The Skype phone application is using the ports 80 and 443:-(.
