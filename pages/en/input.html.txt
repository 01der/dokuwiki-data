===== 2D / 3D Input Devices =====
==== Kinect ====
  *[[http://en.wikipedia.org/wiki/Kinect|EN.Wikipedia "Kinect"]], [[http://de.wikipedia.org/wiki/Kinect|DE.Wikipedia "Kinect"]].
  *Lighting is not needed.
  *The Windows device is equipped with advanced features, in comparison to the original X-Box device.
  *The device might be used for low-budget motion capture :-).
==== Leap Motion ====
  *[[http://www.leapmotion.com/|Leap Motion]].
  *[[http://developer.leapmotion.com/|Leap Motion - Developer Portal]].
  *Blog [[http://blog.leapmotion.com/|Leap Motion Blog]].
  *Twitter.
    *[[http://www.twitter.com/leapmotion|Twitter "LeapMotion"]].
    *[[http://twitter.com/LeapMotionDev|Twitter "Leap Motion Dev"]].
  *[[http://en.wikipedia.org/wiki/Leap_Motion|EN.Wikipedia "Leap Motion"]].
  *Lighting is mandatory.
  *High usage of CPU power, not suitable for devices with limited CPU power ( like webbooks ).
  *The device driver for Win and MacOSX ( but not available for Linux ) is proprietary and just provides raw data. Data processing must be processed by the applications.
==== Duo ====
  *[[http://www.duo3d.com/|Code Laboratories, Inc. "Duo"]].
  *Similar hardware & software concept as "Leap Motion", but different marketing concept and different vendor's vision.
