===== Tessel, Tessel 2 =====
  *[[http://tessel.io/|Tessel]] - "Build your product faster. Tessel 2 is a development platform you can embed in a product. Build fast with Node.js/io.js, then optimize the hardware and build thousands".
    *[[http://github.com/tessel/|GitHub "tessel"]].
      *[[http://github.com/tessel/t1-docs|GitHub "tessel/t1-docs"]] - "API documentation for the original Tessel".
    *[[http://start.tessel.io/install|Tessel "Tessel Installation"]] for Tessel1.
    *[[http://www.tessel.io/modules|Tessel "Modules"]].
      *"Camera"
        *"Note that the Camera Module will not work with Tessel 2. For camera on Tessel 2, use the USB Camera Module".
        *The camera is recording the photos mirror-inverted, as the original CCD hardware does. With usual cams, the correction ( = mirroring back to a "normal" photo ) is done by internal camera firmware.
      *"USB Camera" - "USB Camera Module for Tessel 2. Camera, webcam, video, audio, and still images all wrapped up and supported in one. Should work with any USB webcam and/or USB speakers".
    *As of 2016-02:
      *Tessel1 is incompatible to "NodeJS 4" and so requires to install [[http://www.nodejs.org/|NodeJS]] 0.12.10".

===== WunderBar =====
  *The German Company [[http://www.relayr.io/|iThings4U GmbH "relayr - bring things to life"]]
    *[[http://www.relayr.io/wunderbar|iThings4U GmbH - relayr "WunderBar"]].
    *[[http://developer.relayr.io/|relayr "Get started with Relayr app development"]].
      *[[http://developer.relayr.io/documents/Python/Introduction|relayr "relayr for Pythonists"]] - "relayr Python Library Documentation".
      *[[http://www.github.com/|GitHub "relayr"]].
        *[[http://www.github.com/relayr/python-sdk|GitHub "relayr/python-sdk"]].
      *[[http://pypi.python.org/pypi/relayr/|Python Package Index "relayr"]].
      *The "relayr Python Library" can't be installed on Windows. The installation "pip install relayr" on Python 2.7 aborts by the error message "Pexpect is intened for UNIX-like operating systems" :-(.
        *[[http://pypi.python.org/pypi/pexpect/|Python Package Index "pexpect"]] for Linux.
        *[[http://pypi.python.org/pypi/winpexpect/|Python Package Index "winpexpect"]] for Windows.
    *[[http://www.hackthehouse.io/|relayr Hack The House: With Bosch and Siemens Home Appliances]] by [[http://www.relayr.io/|iThings4U GmbH "relayr - bring things to life"]] and [[http://www.bsh-group.com/|BSH Hausgeräte GmbH]], 2015-02-27 - 2015-03-01.
      *[[http://www.facebook.com/BoschHomeDeutschland/posts/10152548308606556|Facebook "Bosch Home Deutschland: relayr Hack The House"]].
      *[[http://www.facebook.com/groups/455546924601515/|Facebook, Closed Group "relayr Hack The House"]].
    *Business model of "relayr": The Wunderbar devices are communicating by MQTT with cloud services provided by "relayr".
