  *Popular users:
    *The "new" [[http://www.studivz.net/|StudiVZ]].
    *Expert: [[http://www.twitter.com/cainvommars|Twitter "Peter Willert"]], [[http://www.devbliss.com/|devbliss GmbH]] - BarCamp Hamburg 2012 session "GWT für Frontender".
  *Principles: 
    *You write Java code, the GWT creates Javascript for Browsers.
    *You write a [[http://en.wikipedia.org/wiki/Single-page_application|Single-Page Applications]]. There is a file "nocache.js" loaded by the first and only HTML page. Then all other contents is loaded dynamically by this Javascript code.
    *You might distribute the Javascript code generated by GWT by a [[http://www.phonegap.com/|Phonegap]] application.
    *You might communicate in Javascript by REST protocol with server.
