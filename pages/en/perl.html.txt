      *Do not download the archive file as there is no simple automatic installation procedure for this.
      *Instead, just call for online download and automatic online installation :-):
        *Installation with [[http://www.strawberryperl.com/|Strawberry Perl]] on Windows and Perl on Linux:<code>
perl -MCPAN -e shell
install Mojolicious</code>
        *Installation with [[http://www.activestate.com/activeperl/|ActivePerl]] on Windows:<code>
ppm</code> or <code>
ppm -shell</code>
    *Start your first script:
      *Call in your personal home directory your first self-written Embedded Perl script "hello.pl" by <code>
perl hello.pl daemon</code>
      *Not recommended alternative: With COMMAND.COM, go from the root installation directory to the subdirectory "script". There you can run your first self-written Embedded Perl script "hello.pl" by "perl morbo hello.pl".

==== Teaching "use strict" Pragma from Start ====
=== Statement ===
  *I was told by Perl experts, that some old and current Perl books and free Perl online tutorials don´t teach Perl the right way as the autor:
    *Doesn't introduce the "use strict" pragma in the first chapter.
    *Tries to teach Perl by supplying Perl code samples without "use strict" pragma which are not executable anymore if you include the statement :-(.
