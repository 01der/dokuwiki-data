===== Composing & Music/Sound Generation =====
  *The OpenSource [[http://www.sonic-pi.net/|Sonic Pi]] for Windows, Linux on Raspberry Pi, MacOSX ( but not plain Desktop Linux :-( ) - "Audible Computing".
    *[[http://www.raspberrypi.org/learning/sonic-pi-lessons/|Raspberry Pi Learning Resources "Sonic Pi Lessons"]].
    *[[http://www.github.com/samaaron/sonic-pi|GitHub "samaaron/sonic-pi"]] - "The Live Coding Synth for Everyone".
    *The online service [[http://www.sonicpiliveandcoding.com/|Sonic Pi: Live & Coding]].
  *The free online service [[http://www.pencilcode.net/|PENCIL Code]] - "Dream it. Code it. Learn professional programming languages using an editor that lets you work in either blocks or text. Create art, music, games, and stories".
    *[[http://gym.pencilcode.net/jam/|Pencil Code Gym "JAM - Code your own music]].
