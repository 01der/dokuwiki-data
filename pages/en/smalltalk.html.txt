===== Free Smalltalk Development Systems =====
  *[[http://en.wikipedia.org/wiki/Smalltalk|EN.Wikipedia "Smalltalk"]], [[http://de.wikipedia.org/wiki/Smalltalk-80_%28Programmiersprache%29|DE.Wikipedia "Smalltalk-80 (Programmiersprache)"]].
  *The Opensource [[http://www.squeak.org/|Squeak Smalltalk]] for Windows and Linux.
    *[[http://wiki.squeak.org/squeak/1762|SqueakNOS]].
  *The free Smalltalk/V for Win 3.1 .. Win98SE (16-bit application).
    *"This version of Smalltalk Express will not run correctly with more than 256 colors. Please set your video driver to 256 color mode".
    *This software does not run properly on W2k or newer versions ?!
    *[[http://www.andreadrian.de/smalltalk/|Andre Adrian - Smalltalk]] - download of "Smalltalk Express 2.04".
    *[[http://www.smalltalk.org/versions/SmalltalkExpress.html|Smalltalk.org "Smalltalk Express by ObjectShare"]].
  *The free [[http://www.cincomsmalltalk.com/main/products/visualworks/download/|Cincom Smalltalk Non-Commercial Version]] for Windows, Linux - "includes both ObjectStudio 8.2 and VisualWorks 7.7".
    *Just if you need it, Cincom still sells "Visual Smalltalk Enterprise" (VSE), the successor of Smalltalk/V.
    *UML designer.
      *"ADvance" - The integrated UML modeler of VisualWorks.
      *"Modelling Tool" - The integrated UML modeler of ObjectStudio.
    *Documentation.
      *VisualWorks - in the directory "doc". Introductory example "WalkThrough".
      *ObjectStudio - in the directory "OS Manuals".
  *The free [[http://www.exept.de/de/products/smalltalk-x/|eXept Software AG "Smalltalk/X"]] for Windows, Linux.
    *[[http://www.exept.de/de/products/smalltalk-x/stx-sampleapplications|eXept Software AG - Smalltalk/X Sample Applications]].
    *[[http://www.exept.de/de/products/smalltalk-x/stx-download/stx-installation-instructions|eXept Software AG - Smalltalk/X Installation Instructions]] - tips how to organize the filesystem to build executable applications.
  *[[http://smalltalk.gnu.org/|GNU Smalltalk]].
  *[[http://st-www.cs.illinois.edu/archive/|University of Illinois at Urbana Champaign (UIUC) "Index of Smalltalk Archive by Smalltalk Version"]].
