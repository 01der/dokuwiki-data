==== What You must teach an Rookie to master LibreOffice Text / Microsoft Office for Job Applications ====
  -Document size ( Letter vs. A4 ).
  -Fonts ( ARIAL -> CV, Times New Roman ->Newspaper, COURIER -> computer code )
  -Styles ( "Apply Styles", "Font Name", "Font Size" ).
  -Visible and hidden Tables, "Table Format", "Table-Table Boundaries".
  -Table of contents.
  -Resizing of images, "Keep Ratio", image size reduction with GIMP, Paint.NET, Microsoft Expression.
  -"Footnote", "Endnote".
  -Using a simple text editor ( e.g. Notepad on Windows ) as copy+paste buffer, to get style-free text for import.
  -Strategy to clone a document, i.e. to exchange text and graphics, instead of building a document from scratch.
  -Check the document by:
    -Creating a PDF.
    -Do a visual inspection of the PDF.
  -Hard page break ( ctrl+return ).
