=== Import of Stencils and Templates ===
    *Import of Stencils:
      *"Tool / Install New Collection..."  aborts with the error message "Error installing collection" :-(.
      *[[http://groups.google.com/forum/#!topic/pencil-user/AMTt7EbExdU|Google Groups "Pencil Users", Thread "Impossible to import new stencil collections"]], [[http://code.google.com/p/evoluspencil/issues/detail?id=133|Google Code "evoluspencil. The open-source GUI prototyping tool for GNU/Linux, Windows and Mac" - Issue 133]], [[http://code.google.com/p/evoluspencil/issues/detail?id=586|Google Code "evoluspencil. The open-source GUI prototyping tool for GNU/Linux, Windows and Mac" - Issue 586]] -> Workaround:
        *Extract the archives in "<PencilProfile>/Pencil/Stencils" directory.
        *For the standolone (xulrunner) version:
          *By example, on win7 by default: "%USERPROFILE%\AppData\Roaming\Pencil\Profiles\{your-profile-name}\Pencil\Stencils"
          *On my computer "C:\Users\Administrator\AppData\Roaming\Pencil\Profiles\kei9cefg.default\Pencil\Stencils" ( The profile name "kei9cefg.default" was created by the application, not by my own will ).
        *For the Firefox extension version:
          *By example, on win7 by default: "%USERPROFILE%\AppData\Roaming\Mozilla\Firefox\Profiles\{your-profile-name}\Pencil\Stencils"
        *Each Stencil has its own a directory in this directory, with a "Definition.xml" file (and some other optional files).
          *By example:
            *"[...]\Pencil\Stencils\ExtJs.GUI_v11\Definition.xml".
            *"[...]\Pencil\Stencils\ExtJs.GUI_v11\Icons\[...]".
    *Import of Templates:
      *"Tools / Manage Export Template...", button "Install New Template...".
      *If a template is already installed, there is the error message "Error installing template".

===== Prototype =====
==== Features ====
  *Navigation, Visual Elements, Interactivity, Functionality.
  *High fidelifty interface that allows for aesthetic design and basic interactivity. Usually associated with clickable, dynamic mockups. Typically focus on behavior and advanced function.
