==== (Linux)ISO-2-USB ====
  *During installation of Linux to boot from USB, please set "Persistence = 0 to x Mbytes" for successful operation on computers with SDD disks, e.g. ASUS Eee PC 901.
  *You must enable the "Autorun" function ( in the settings of security and antivirus software like [[http://www.avira.com/de/avira-free-antivirus|Avira Free Antivirus]] ) for proper operation of this software. Or you must disable the security and antivirus software totally for operation of ISO-2-USB transfers.
