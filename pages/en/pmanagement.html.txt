===== Scrum, Agile Development Extreme Programming / Xtreme Programming =====
==== Local Optimization vs. Global Optimization ====
  *Agile methods -> local optimization.
  *Traditional planning methods -> global optimization.
==== Mandatory ( must-have ) Requirements for using Scrum in a Project ====
=== Maintainability ===
  *Oldfashioned Opinion: "The software must be highely maintainable".... but the software code isn't maturing and can´t get corrupted ( in opposite to physical objects ), as it is the same code all the time.
    *[[http://en.wikipedia.org/wiki/Maintainability|EN.Wikipedia "Maintainability"]], [[http://de.wikipedia.org/wiki/Wartbarkeit|DE.Wikipedia "Wartbarkeit"]].
=== Evolvability ===
  *Modern Opinion: "The software must have a high evolvability".
    *[[http://de.wikipedia.org/wiki/Evolvierbarkeit_%28Software%29|DE.Wikipedia "Evolvierbarkeit (Software)"]].
