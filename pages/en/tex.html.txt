===== LaTeX Software Distributions =====
  *MiKTeX ( [[http://www.miktex.org/|MiKTeX]], [[http://miktex.sourceforge.net/|MiKTeX]], [[http://www.sourceforge.net/projects/miktex/|MiKTeX]] ).
  *The [[http://www.ctan.org/|Comprehensive TeX Archive Network]] ( CTAN ).
    *[[http://tug.ctan.org/cgi-bin/ctanPackageInformation.py?id=miktex-portable|A version of MikTeX to carry around]].
  *[[http://tug.org/texlive/|TeX Live]].
  *German language support is provided by the package [[ftp://ctan.unsw.edu.au/tex-archive/systems/win32/miktex/tm/packages/german.cab|german.cab]]. It is not included in the MiKTeX distribution !
  *Package [[ftp://ctan.unsw.edu.au/tex-archive/systems/win32/miktex/tm/packages/koma-script.cab|scrreprt]].
  *The commercial [[http://www.mackichan.com/|MacKichan Software - "Scientific WorkPlace", "Scientific Word", "Scientific Notebook"]].

===== LaTeX Tools ===== 
  *bm2font.
    *[[ftp://ftp.uni-duesseldorf.de/pub/tex/bm2font/|Official FTP dowload of bm2font]].
    *[[http://gnuwin32.sourceforge.net/packages/bm2font.htm|Sourceforge: bm2font]].
    *[[http://ctan.cdpa.nsysu.edu.tw/graphics/bm2font/|ctan dowload of bm2font.]] - extra win32 version of bm2font, of 2000.
  *[[http://www.informatica.us.es/~calvo/latex_en.html|Tablas]] by Pedro L. Luque - "program that allows to export and to import tables Latex of visual form".

    *[[http://www.ctan.org/tex-archive/support/latable/|LaTable]] - "A near-WYSIWYG editor for LaTeX tables".
  *[[http://www.word2tex.com/|Chikrii Softlab ]] - the Shareware "TeX2Word" and "Word2TeX" tools.
  *JabaRef ( [[http://jabref.sourceforge.net/|JabaRef]], [[http://www.sourceforge.net/projects/jabref/|JabaRef]] ).
