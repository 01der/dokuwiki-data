===== Risk Management =====
==== Fault Tree Analysis ====
  *[[http://en.wikipedia.org/wiki/Fault_tree_analysis|EN..Wikipedia "Fault tree analysis"]], [[http://de.wikipedia.org/wiki/Fehlerbaumanalyse|DE.Wikipedia "Fehlerbaumanalyse"]].
  *Software fault tree analysis ( SWFTA ) - A deductive analysis: It starts with potential or actual failures and asks what might have caused them.
==== Failure Mode and Effects Analysis ( FMEA ) / Failure Mode and Effects and Criticality Analysis ( FMECA ) ====
  *[[http://en.wikipedia.org/wiki/Failure_mode_and_effects_analysis|EN.Wikipedia "Failure mode and effects analysis"]], [[http://de.wikipedia.org/wiki/FMEA|DE.Wikipedia "FMEA"]].
  *Software failure mode, effect and criticality analysis ( SWFMECA ) - An inductive analysis: Tt starts with the failure of components and asks what they might result in at the systems level.

  *[[http://www.markovanalysis.com/|Isograph Ltd "Markov Analysis"]].

  *[[http://en.wikipedia.org/wiki/Bayes%27_theorem|EN.Wikipedia "Bayes' theorem"]], [[http://de.wikipedia.org/wiki/Bayestheorem|DE.Wikipedia "Bayestheorem"]].
