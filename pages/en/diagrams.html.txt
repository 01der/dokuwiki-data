===== Tools for Database Application Design by Data Flow Diagrams ( DFD ) =====
==== Free Tools specially for Database Application Design by Data Flow Diagrams ( DFD ) ====
  *The free [[http://www.klassenkarte.de/?page_id=15|Klassenkarte "Orinoco"]] for WinXP with .NET 4.0 - "Bei Orinoco handelt es sich um ein Programm, mit dem man Datenflussdiagramme entsprechend dem Eingabe-Verarbeitung-Ausgabe- (EVA-) Prinzip modellieren und auswerten kann.
